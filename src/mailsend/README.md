# Mail

Mime Message creation and smtp sending

## USER

- create user with 

php vendor/bin/cryodrift.php /user/cli register -user="SAMEUSERNAME" -password

php vendor/bin/cryodrift.php -sessionuser="SAMEUSERNAME" /user/cli mailaccount -type="" -host="smtp.hostname" -name="mailusername" -pass=""
